HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element also contains a "header", "main" and "footer".
  • Inside of these elements there are further elements used to organize your code.
    • h1, h2, h3, h4, h5, h6: headings for sections the number denoting it's degree of importance
    • img: an image this will contain a href to an image file hosted elsewhere
    • section: a way to seperate different parts of you content diplayed
    • ul: unordered list containing "li" these lists are those in which the order of the "li" is not needed to be understood
    • ol: ordered list also containing "li" where the order is very important (ex. recipe)
    • li: list item found inside eith "ul" or "ol"
    • span
    • aside
    • p: paragraph for longer chunks of text content
  • Within these elements we can have classes or id's to target them for further edit in CSS and JavaScript

Git

  • git status: chcks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "commit-message": saves all changes made locally
  • git push: pushes all locally commited changes to the remote branch
  • git pull: will fetch all changes from remote repo and merge remote change in the current local branch
